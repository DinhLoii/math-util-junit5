# Math-Util Library

## Overview
**Math-Util** is a Java utility library providing common mathematical functions.  
The project demonstrates **unit testing**, **regression testing**, and **Continuous Integration (CI)** using **GitHub Actions**.

---

## Key Features
- Java-based math utility library
- Unit testing with **JUnit 5**
- Automated build & test with **Maven**
- Continuous Integration using **GitHub Actions**

---

## Tech Stack
Java (JDK 21) · Maven · JUnit 5 · GitHub Actions
